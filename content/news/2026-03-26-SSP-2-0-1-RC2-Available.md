# Release Candidate for SSP 2.0.1

The second release candidates for the maintenance releases [SSP 2.0.1](https://github.com/modelica/ssp-standard/releases/tag/v2.0.1-rc.2) is ready for final review.

According to the procedure for the release of maintenance releases of MA standards, a 4-week inspection period starts and will last until April 26, 2026.
In the last week of the inspection period, only editorial or typographic changes are allowed.
In case of findings, please open an issue on https://github.com/modelica/ssp-standard.
If there are no blockers detected, the SSP Project will release SSP 2.0.1 afterwards.
